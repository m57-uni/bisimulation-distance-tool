# Bisimulation Distance Tool

in probabilistischen Transitionssystemen.

**Kontext:** Bachelorarbeit, Lehrstuhl Theoretische Informatik, 
Universität Duisburg-Essen, 2023.

**Stack:** Python, SciPy, NumPy
